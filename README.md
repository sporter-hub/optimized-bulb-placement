# optimized-bulb-placement
Identified optimal bulb distance (d = 1.44m) using root-finding command (fzero) to maximize illumination in the room's darkest spots. Achieved a minimum intensity of 0.93W/m^2, representing a 50% increase in efficiency compared to single-bulb setups.
